Cyber Security

This section of my portfolio showcases hands-on cybersecurity projects focused on offensive security, digital forensics, cloud security, network analysis, vulnerability assessment, security monitoring, and risk management. Through practical experience with industry-standard tools such as Kali Linux, Wireshark, Burp Suite, OWASP ZAP, Splunk, AWS, Autopsy, Nmap, and DVWA, I have developed skills in identifying vulnerabilities, analyzing network and system activity, conducting security assessments, investigating digital evidence, and implementing security controls. These projects demonstrate my ability to apply cybersecurity concepts in real-world scenarios, covering the complete security lifecycle from threat detection and vulnerability exploitation to incident analysis, cloud protection, and security monitoring.
